# 簡介
* OpenMAX是一個多媒體應用程序的標準。無授權費、跨平臺C語言程序接口序列，這些接口對音頻、視頻、靜態圖片的常用操作進行封裝。
* 應用層（AI）、集成層（IL）和開發層（DL）
# 在Android中的位置
OpenMax IL層，通常可以用於多媒體引擎的插件，Android的多媒體引擎OpenCore和StageFright都可以使用OpenMax作爲插件
